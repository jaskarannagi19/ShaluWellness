Introduction


Dhyana - Yoga Studio & Meditation HTML Template

Dhyana is a clean, modern and mobile-ready Yoga Studio & Meditation website Template. Whether you are starting up a yoga or you wish to establish awareness for a Yoga Studio, this Yoga Studio website template can help you find the most appropriate web design and layout for your taste. Specially for Yoga Studios, Trainers, Spas, Fitness, Beauty, Yoga, Salons, Massage, Care, Hair, Health, Wellness Center and personal website.

Dhyana is based on the popular Bootstrap Framework that allows flexibility and extendability. Take your yoga business to the next level, with Dhyana Yoga Studio!

Configuration and Installation

Dhyana-HTML-Package zip: Main zip file HTML and Documentation Included

Dhyana-HTML-Package.zip: Dhyana-HTML-Package.zip file extract and main HTML template and documentation folder are included.

(1) Dhyana-HTML and (2) Documentation folder

Open the "Dhyana-HTML" folder and run index.html in your browser

  1. Open the Dhyana-HTML Folder to find all the Templates Files
  2. You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  3. Make sure you upload the required files/folders listed below:
    • Dhyana-HTML/assets/bootstrap - Bootstrap Stylesheets Folder
    • Dhyana-HTML/assets/css - Extra Stylesheets Folder
    • Dhyana-HTML/assets/fonts - Fonts Folder
    • Dhyana-HTML/assets/images - Images Folder
    • Dhyana-HTML/assets/js - Javacripts Folder
    • Dhyana-HTML/assets/owlcarousel - owlcarousel Stylesheet File
    • Dhyana-HTML/assets/index.html - Index File/Homepage
    The other files can be used according to your preferences.
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

FAQ



How to remove the top menu sticky as scrolling?

  • Open Up your HTML file 'Dhyana-HTML/' file location. (For Example, 'index.html'
  • Then Search For the Comment '<!-- START HEADER -->'
  • show this line - <header class="header_wrap">
  • add class name - <header class="header_wrap no-sticky"> (add class name 'no-sticky')

Where i can define my e-mail account to receive the messages from contact form?

  • Open Up your HTML file 'Dhyana-HTML' file location. (For Example, 'Dhyana-HTML/contact.php'
  • show this line number 11 - $mailTo = 'email@example.com'
  • 'email@example.com' enter your email account to receive the messages from contact form

How to Change Color scheme?

  • Open Up your HTML file 'Dhyana-HTML' file location. (For Example, 'Dhyana-HTML/index.html'
  • show this line number 40 - <link rel="stylesheet" id="layoutstyle" href="assets/color/theme-default.css">
  • Use this - theme-blue-light, theme-green, theme-orange, theme-pink

How to Use Boxed Layout?

  • Open Up your HTML file 'Dhyana-HTML' file location. (For Example, 'Dhyana-HTML/index.html'
  • Find the <body> tag and add class boxed_layout

How to Use Background Image in Boxed Layout?

  • Open Up your HTML file 'Dhyana-HTML' file location. (For Example, 'Dhyana-HTML/index.html'
  • Find the <body> tag and add style="background-image: url("assets/images/layout_bg1.jpg");"
  • Use this layout_bg1, layout_bg2, layout_bg3, layout_bg4, layout_bg5, layout_bg6
